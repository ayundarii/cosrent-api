# Cosrent API

This is the backend API for the Cosrent project, a cosplay rental platform.

## Installation

1. Install all dependencies:
    ```npm install```

2. Generate Prisma:
    ```npx prisma generate```

3. Run Prisma migrations:
    ```npx prisma migrate dev```

4. Run the seeder:
    ```npm run seed```

5. Run the project:
    ```npm run dev```
